# Python Farming Ürün Vizyonu

Python Farming; kullanıcıyı pasif video izleyicisinden aktif Python geliştiricisine dönüştüren, tamamen kod ve uygulama üzerinden ilerleyen masaüstü öğrenme oyunudur.

## Temel ilke

Kullanıcı önce yönlendirilmiş kısa görevlerle öğrenir, ardından pratik modlarına, hata ayıklama görevlerine, mini projelere ve çok dosyalı Expert Lab projelerine geçer.

## Görsel yön

- Assetsiz
- Masaüstü verimliliği odaklı
- Koyu kömür yüzeyler
- Tek vurgu rengi: amber
- İnce ayırıcılar
- Az gölge ve az radius
- Kod editörü merkezli
- Çocuk oyunu veya jenerik SaaS görünümünden uzak
